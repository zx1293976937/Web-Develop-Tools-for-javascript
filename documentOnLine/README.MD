概述
这是一个基于nodejs+express(ejs)+jquery+milk的Web单页应用, 用于展示milk框架的各个函数用法

源代码管理(开源)
https://github.com/WellerQu/Web-Develop-Tools-for-javascript/

软件环境
nodejs

操作系统
Window 7 / Ubuntu 14 / Mac OS 10.9

启动
1. 在命令行跳转到应用根目录
2. 执行npm install 命令
3. 执行npm start 命令
4. 在浏览器地址栏输入 http://localhost:3000/ 访问
